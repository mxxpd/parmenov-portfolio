# Parmenov Portfolio

Персональное портфолио Максима Парменова на `Nuxt 4` с главной страницей, кейсами, контактной модалкой, сервисами и статьями.

Проект в активной разработке: базовая архитектура, дизайн-система, маршруты, адаптивы и первый подробный кейс уже есть, контент и разделы продолжают развиваться.

## Features

- Главная с адаптивным hero-блоком и CTA.
- Страница кейсов с микро-лоадером при переходе в кейс.
- Подробный кейс `Your Challenge`, собранный из секций.
- Глобальная контактная модалка с валидацией, success-состоянием и a11y-фокусом.
- Переиспользуемые UI-компоненты: `AppButton`, `AppInput`, `AppSelect`, `AppTextarea`, `AppTag`.
- Светлая/тёмная темы и кастомные скроллбары.
- Мобильное меню, которое открывается под хедером и не перекрывает управление.

## Stack

- `Nuxt 4`
- `Vue 3`
- `TypeScript`
- `SCSS` (sass-embedded)
- `Typograf`
- `vite-svg-loader`

## Development

Установка зависимостей:

```bash
npm install
```

Запуск через Docker (основной режим):

```bash
npm run dev
```

Локальный запуск без Docker:

```bash
npm run dev:local
```

Остановка:

```bash
npm run dev:down
```

## Build

```bash
npm run build
```

## Quality

```bash
npm run lint
npm run typecheck
```

## Routes

- `/` — главная
- `/cases` — список кейсов
- `/cases/your-challenge` — подробный кейс
- `/services` — заглушка раздела сервисов
- `/articles` — заглушка раздела статей
